## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/yourusername/cppmcp.git`
3. **Create a branch**: `git checkout -b feature/your-feature`
4. **Make changes** and commit: `git commit -m 'Add amazing feature'`
5. **Push** to your fork: `git push origin feature/your-feature`
6. **Create a Pull Request**

## Code Style

- Use C++17 features
- Follow existing code formatting
- Add comments for complex logic
- Update documentation when needed

## Testing

- Add tests for new features
- Ensure all tests pass: `cmake --build build && ctest --test-dir build`
- Test both STDIO and SSE modes

## Commit Messages

- Use clear, descriptive messages
- Reference issues when applicable
- Examples:
  - `feat: Add new tool registration API`
  - `fix: Resolve memory leak in SSE transport`
  - `docs: Update README examples`

## Questions?

Open an issue or start a discussion. We're happy to help!
